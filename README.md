# iPhone Sales Data Analysis

## Project Overview

This project analyzes iPhone sales data using Python to better understand sales trends, customer preferences, and product performance. The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization to uncover meaningful insights from the dataset.

The project demonstrates practical data analysis techniques and highlights how data can be used to support business decisions.

## Objectives

- Explore the iPhone sales dataset.
- Clean and preprocess the data.
- Handle missing values and duplicate records.
- Perform exploratory data analysis (EDA).
- Visualize important sales trends.
- Extract meaningful business insights.

## Tools and Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

1. Imported the dataset.
2. Explored the dataset structure.
3. Cleaned and prepared the data.
4. Performed exploratory data analysis.
5. Created visualizations to identify sales patterns.
6. Summarized key findings and insights.

## Key Insights

- Analyzed iPhone sales performance.
- Explored customer ratings and reviews.
- Identified pricing and sales trends.
- Examined relationships between different variables.
- Created visualizations to support data-driven insights.
